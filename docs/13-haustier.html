Haustier-Tracking: Charly

Charly ist unsere Katze und wird über zwei Integrationen in Home Assistant getrackt: Tractive für GPS-basiertes Outdoor-Tracking und PetKit für die Überwachung von Fütterung und Aktivität im Haus. Zusammen ergeben diese Integrationen ein umfassendes Bild über Charlys Aufenthaltsort, Aktivitätsmuster und Wohlbefinden.

Das Charly Dashboard mit GPS-Tracking, Fütterungsdaten und Aktivitätsübersicht

Tractive GPS Tracker

Der Tractive GPS Tracker ist ein kleines, am Halsband befestigtes Gerät, das Charlys Position in Echtzeit über Mobilfunk an die Tractive-Cloud übermittelt. Die HACS-Integration bringt diese Daten direkt in Home Assistant.

Integration einrichten

  1. Tractive GPS Tracker am Halsband befestigen und in der Tractive App aktivieren
  2. In HACS nach Tractive suchen und installieren (oder native Integration nutzen)
  3. Home Assistant neu starten
  4. Einstellungen → Geräte & Dienste → Integration hinzufügen
  5. Nach Tractive suchen
  6. E-Mail und Passwort des Tractive-Kontos eingeben
  7. Die Integration erkennt automatisch alle registrierten Tracker

Verfügbare Entities

Entity Typ Beschreibung
device_tracker.charly Device Tracker GPS-Position mit Latitude/Longitude
sensor.charly_battery_level Sensor Akkustand des Trackers in Prozent
sensor.charly_activity Sensor Aktuelle Aktivität (Ruhen, Aktiv, etc.)
sensor.charly_speed Sensor Aktuelle Geschwindigkeit
binary_sensor.charly_tracker_online Binär-Sensor Tracker-Verbindungsstatus

Device Tracker Attribute

Die device_tracker.charly Entity enthält folgende Attribute: 

# device_tracker.charly - Attribute
source_type: gps
latitude: 51.xxxxx
longitude: 7.xxxxx
gps_accuracy: 15          # Genauigkeit in Metern
altitude: 85              # Höhe über NN
battery_level: 78         # Akkustand in %
speed: 0.0                # Geschwindigkeit in km/h
course: 180               # Bewegungsrichtung in Grad
friendly_name: "Charly"
entity_picture: "/local/charly.jpg"

Zonen für Charly

Um zu erkennen, ob sich Charly in bestimmten Bereichen aufhält, werden Home Assistant Zonen definiert. Der device_tracker ändert seinen Status automatisch basierend auf diesen Zonen. 

# configuration.yaml - Zonen
zone:
  - name: "Zuhause"
    latitude: !secret home_latitude
    longitude: !secret home_longitude
    radius: 50
    icon: mdi:home

  - name: "Garten"

    longitude: !secret garden_longitude
    radius: 30
    icon: mdi:flower

  - name: "Nachbar Links"

    longitude: !secret neighbor_left_longitude
    radius: 25
    icon: mdi:home-group

PetKit Integration

Die PetKit Integration bindet smarte Haustiergeräte ein, wie beispielsweise automatische Futterautomaten und Trinkbrunnen. Sie liefert Daten über Fütterungszeiten, Futtermengen und den Gerätestatus.

Integration einrichten

  1. PetKit-Geräte über die PetKit App einrichten und mit dem Konto verbinden
  2. In HACS nach PetKit suchen und installieren
  3. Home Assistant neu starten
  4. Einstellungen → Geräte & Dienste → Integration hinzufügen
  5. PetKit Zugangsdaten eingeben

Verfügbare Entities

Entity Typ Beschreibung
sensor.petkit_feeder_food_level Sensor Füllstand des Futterautomaten
sensor.petkit_feeder_last_feeding Sensor Zeitpunkt der letzten Fütterung
sensor.petkit_feeder_daily_amount Sensor Heutige Futtermenge in Gramm
button.petkit_feeder_dispense Button Manuelle Fütterung auslösen
sensor.petkit_fountain_water_level Sensor Wasserstand des Trinkbrunnens
sensor.petkit_fountain_filter_life Sensor Restlebensdauer des Filters in Prozent

Charly Dashboard

Das Charly Dashboard bietet eine zentrale Übersicht über alle Haustier-relevanten Daten: Standort auf einer Karte, Aktivitäts-Verlauf, Tracker-Akkustand, Fütterungsdaten und den Status der PetKit-Geräte.

Dashboard-Konfiguration

# dashboards/haustier.yaml
views:
  - title: Charly
    icon: mdi:cat
    cards:
      # Karte mit Charlys Position
      - type: map
        title: "Charly - Standort"
        entities:
          - entity: device_tracker.charly
        default_zoom: 17
        hours_to_show: 24
        dark_mode: true
        aspect_ratio: 16:9

      # Status-Übersicht
      - type: entities
        title: "Charly - Status"
        entities:
          - entity: device_tracker.charly
            name: Aufenthaltsort
          - entity: sensor.charly_activity
            name: Aktivität
          - entity: sensor.charly_battery_level
            name: Tracker Akku
          - entity: sensor.charly_speed
            name: Geschwindigkeit
          - entity: binary_sensor.charly_tracker_online
            name: Tracker Online

      # Tracker-Akku Gauge
      - type: gauge
        entity: sensor.charly_battery_level
        name: "Tracker Akku"
        min: 0
        max: 100
        severity:
          green: 50
          yellow: 20
          red: 0

      # Fütterung
      - type: entities
        title: "Fütterung"
        entities:
          - entity: sensor.petkit_feeder_food_level
            name: Futterfüllstand
          - entity: sensor.petkit_feeder_last_feeding
            name: Letzte Fütterung
          - entity: sensor.petkit_feeder_daily_amount
            name: Tägliche Menge
          - entity: button.petkit_feeder_dispense
            name: Jetzt füttern

      # Trinkbrunnen
      - type: entities
        title: "Trinkbrunnen"
        entities:
          - entity: sensor.petkit_fountain_water_level
            name: Wasserstand
          - entity: sensor.petkit_fountain_filter_life
            name: Filter Restlebensdauer

      # Aktivitätsverlauf (Grafik)
      - type: history-graph
        title: "Aktivitätsverlauf (24h)"
        hours_to_show: 24
        entities:
          - entity: sensor.charly_activity
          - entity: device_tracker.charly

Automatisierungen für Charly

Charly verlässt den Garten

- alias: "Charly verlässt den Garten"
  description: "Benachrichtigung wenn Charly die Gartenzone verlässt"
  trigger:
    - platform: zone
      entity_id: device_tracker.charly
      zone: zone.garten
      event: leave
  condition:
    - condition: state
      entity_id: input_boolean.charly_tracking_aktiv
      state: "on"
  action:
    - service: notify.alexa_media_marcus_echo_show_8_2nd_gen
      data:
        message: "Charly hat den Garten verlassen."
        data:
          type: tts
    - service: notify.mobile_app_marcus_handy
      data:
        title: "Charly unterwegs"
        message: >
          Charly hat den Garten verlassen und befindet sich jetzt bei
          {{ state_attr('device_tracker.charly', 'latitude') }},
          {{ state_attr('device_tracker.charly', 'longitude') }}.
        data:
          actions:
            - action: "OPEN_MAP"
              title: "Auf Karte anzeigen"

Standort-Verlauf

Home Assistant speichert den Standort-Verlauf von Charlys Tracker automatisch in der Recorder-Datenbank. Über das Map-Dashboard kann der Verlauf der letzten Stunden nachverfolgt werden.

Recorder-Konfiguration

# configuration.yaml - Recorder für Standort-Verlauf
recorder:
  include:
    entities:
      - device_tracker.charly
      - sensor.charly_activity
      - sensor.charly_battery_level
      - sensor.petkit_feeder_food_level
      - sensor.petkit_feeder_daily_amount
  commit_interval: 30    # Alle 30 Sekunden speichern

Verlaufsanalyse

Der Standort-Verlauf zeigt typische Bewegungsmuster:

  • Morgens (6-8 Uhr): Charly erkundet den Garten und die nähere Umgebung
  • Vormittags (8-12 Uhr): Meist im Haus oder auf der Terrasse
  • Nachmittags (14-17 Uhr): Längere Ausflüge in die Nachbarschaft
  • Abends (17-22 Uhr): Rückkehr nach Hause, Fütterungszeiten
  • Nachts (22-6 Uhr): Im Haus

Input Boolean für Tracking

# Tracking aktivieren/deaktivieren
input_boolean:
  charly_tracking_aktiv:
    name: "Charly Tracking Benachrichtigungen"
    icon: mdi:map-marker-alert
    initial: true

Tipps

  • Tracker-Akku: Der Tractive Tracker hält je nach Tracking-Modus 2-5 Tage. Im LIVE-Modus entlädt er sich schneller.
  • GPS-Genauigkeit: In Innenräumen ist die GPS-Genauigkeit eingeschränkt. Der Standort kann um mehrere Meter abweichen.
  • Zonen klein halten: Zonen für Katzen sollten eher klein definiert werden (25-50m Radius), da Katzen sich in engen Bereichen bewegen.
  • Benachrichtigungen dosieren: Zu viele Benachrichtigungen können nervig sein. Den Input Boolean nutzen, um Tracking bei Bedarf zu deaktivieren.
  • PetKit Cloud: Die PetKit Integration benötigt eine Cloud-Verbindung. Bei Internetausfällen funktionieren die Automaten weiterhin, aber die Daten kommen nicht in HA an.
